slang
=====
An open-source interpreter for *The Source*.

Usage
-----
The REPL is still under development.

To build,

.. code-block::

  $ git clone https://github.com/source-academy/js-slang.git
  $ cd slang
  $ yarn
  $ yarn build

To invoke the repl, run the built `slang.js`,

.. code-block::

  $ ./slang.js
  >
